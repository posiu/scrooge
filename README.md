# 🤑 Scrooge — Personal Finance Manager

Scrooge to nowoczesna aplikacja do zarządzania finansami osobistymi, która pomoże Ci przejąć pełną kontrolę nad Twoim portfelem. Zapomnij o skomplikowanych arkuszach kalkulacyjnych — Scrooge oferuje prostotę, szybkość i inteligentną analitykę.

## 🎯 Cel Projektu

Stworzenie narzędzia, które pozwala na:
1.  **Agregację wszystkich kont**: Jedno miejsce na stan kont bankowych, gotówki, krypto i inwestycji.
2.  **Śledzenie każdego grosza**: Szybkie dodawanie i kategoryzowanie transakcji.
3.  **Planowanie budżetu**: Definiowanie limitów na kategorie i monitorowanie ich realizacji w czasie rzeczywistym.
4.  **Głęboką analitykę**: Automatyczne raporty miesięczne, trendy wydatków i prognozy finansowe.
5.  **Bezpieczeństwo danych**: Twoje dane finansowe są Twoje — pełna kontrola nad infrastrukturą (Supabase/Postgres).
